The dictionary module

Immutable data objects for JSON serializable key/value pairs.

API

import {
  dictionary,
} from 'https://m.bednarz.dev/dictionary.js';
dictionary(data: object): object
  • create an object without a constructor or prototype
  • copy all string properties of data to that object
  • freeze and return that object
