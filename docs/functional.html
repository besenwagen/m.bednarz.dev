The functional module

Utitlities for functional programming.

Public API

import {
  compose,
  pipe,
  partial,
} from 'https://m.bednarz.dev/functional.js';
compose(...callbacks: function[]): function
pipe(...callbacks: function[]): function
  • return a function that sequentially passes its argument through its callbacks and returns the reduced result
    • compose passes from right to left
    • pipe passes from left to right
partial(callback: function, ...argumentList: any[]): function
  • return a function that returns the return value of callback called with argumentList and its own arguments

Example

Partial application and functional composition: 

const add = (a, b) => a + b;
const multiply = (a, b) => a * b;

const addFive = partial(add, 5);
const timesThree = partial(multiply, 3);
const reduceLeft = pipe(addFive, timesThree);
const reduceRight = compose(addFive, timesThree);

addFive(3) // => 8
timesThree(4) // => 12

reduceLeft(2) // => 21
reduceRight(2) // => 11
