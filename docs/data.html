The data module

Utitlities for JSON data.

API

import {
  normalize,
  serialize,
} from 'https://m.bednarz.dev/data.js';
normalize(data: object|any[]): object|any[]
  • return a deep copy of data with object property names sorted recursively
  • throw a TypeError for values that are not JSON data types
serialize(data: object|any[]): string
  • normalize the passed JSON data
  • return it stringifed
